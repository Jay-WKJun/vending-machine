# 🎯 실행 방법

package manager는 pnpm을 이용해주세요.

```bash
# pnpm이 없는 경우
$npm i -g pnpm

# 패키지 설치
$pnpm i

# 실행
$pnpm dev

# 브라우저로 http://localhost:5173에 접근
```

# ⚙️ 기능 소개

- 번호를 통한 제품 탐색 및 지정
- 자유로운 결제 수단 투입
- **상품과 결제 수단이 준비되면 자동 결제**
- **결제 완료 후 자동으로 제품 배출**
